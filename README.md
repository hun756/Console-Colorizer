# Console-Colorizer

## ~150 lines Ascii Escape Sequnce Console Colorizing.

----------

Supporting Colors:
-  Red     
-  Green   
-  Yellow  
-  Blue    
-  Magenta 
-  Cyan    
-  White

## Usage
